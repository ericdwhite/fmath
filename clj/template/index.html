Intro:What's this about?

This site contains a collection of charts with content generated using Clojure. All the charts on this page are dynamically generated, and the charts can be zoomed. Clicking minimizes a section, and expands.

All of the source code can be found here: Github: ericdwhite/fmath. For issues or comments please drop a note here.

As an example the following is a coin flip random walk.

  • randwalk.clj

Tools and Resources:

There were a couple of tools and resources that were key in building this site and I would like to highlight those:

  • jqPlot by Chris Leonello.
  • ClojureDocs.
  • Compojure and especially this set of tutorials.

Again, the source code can be found here: Github: ericdwhite/fmath
